Using Apache HTTPD

Seeing User Pages

How to set up the local home page on Apache HTTPD.

    HTTPD Settings
  • As root edit /etc/httpd/conf.d/userdir.conf and follow the instructions:-
    • Comment out the "UserDir disabled"
    • Uncomment the "UserDir public_html"
  • Get HTTPD to reload the config 
    systemctl reload httpd
    Selinux Settings
  • Set boolean to allow user home pages 
    setsebool -P httpd_enable_homedirs true
    Directory Settings
  • As yourself:-
  • Create the directory ~/public_html
  • Grant read and execute on that directory 
    chmod og+rx ~/public_html
  • Grant read on your home directory 
    chmod og+r ~
