Enabling Your Site

    Steps to getting your web pages bootstrapped.
  1. Download the zip file from the getting started page of the bootstrap site.
  2. Go to /var/www/html and unzip the file.
  3. (SELinux Only) Assigned the correct label "httpd_sys_content_t" to the new directory. 
    chcon -R -t httpd_sys_content_t /var/www/html/bootstrap-*-dist
  4. Create a soft link "bootstrap" in the same directory to the unzipped directory.
  5. Add this text to the head section. Make sure these declarations come before any of your own declarations. 
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
<link rel="stylesheet" type="text/css" href="/bootstrap/css/bootstrap.min.css">
  6. Add this text to the bottom of the page 
    <script src="http://code.jquery.com/jquery.min.js"></script>
<script src="/bootstrap/js/bootstrap.min.js"></script>

Creating A Layout

    This page has a simple layout with a header (the title and references), some columns (each section) and a footer (the navigation section).

  1. Sorround all of the page that you want to be bootstrapped with a "div" tag with a class of "container". 
    <div class="container">
</div>
  2. With the "container", the header can be picked out by sourrounding it with a div of the "jumbotron" class. 
    <div class="jumbotron">
<h1>Big Heading</h1>
<p>My heading text</p>
</div>
  3. The sections that have equal prominence should be wrapped in a "row" div, and then each section is wrapped in its own "col-xs-*" div. The "*" represents how wide the column should be, between 1 and 12. The maximum number of columns in a row is 12, so by setting these values you govern how these columns rearrange themselves as the width of the browser changes. 
    <div class="row">
<div class="col-xs-4">
<p>Section 1 text.</p>
</div>
<div class="col-xs-4">
<p>Section 2 text.</p>
</div>
<div class="col-xs-4">
<p>Section 3 text.</p>
</div>
</div>
  4. The navigation section uses a "nav" and a "navbar" tag.
