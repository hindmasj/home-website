D3 Joins

Join data based on the Thinking with Joins tutorial, and Scott Murray's D3 Tutorials.

To use, type some numbers into the text box below and hit return. What is produced is a set of bars whose height is proportional to the data, and a set of circles whose x position and area are proportional to the same data. Apart from the form and an anchoring div, all of the elements are created on the fly.
